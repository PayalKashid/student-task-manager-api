# Student Task Manager API

A simple CRUD REST API built using FastAPI.

## Features

- Create student tasks
- View all tasks
- View a single task
- Update tasks
- Delete tasks
- Pydantic validation
- SQLite database
- Swagger documentation

## Technologies Used

- Python
- FastAPI
- Pydantic
- SQLAlchemy
- SQLite
- Uvicorn

## Running the Project

Install dependencies:

pip install -r requirements.txt

Run:

uvicorn main:app --reload

Open:

http://127.0.0.1:8000/docs
